24px icons

libraries will automically choose the right icon

React

- no more width and height, use size prop instead
- icon names have `Icon` at the end
- no more octicon component, use icon components by themeselves

StyledOcticon

- update primer components?
- uninstall @primer/octicons-react
- install @primer/styled-octicons
- udpate usages

```diff
- import {StyledOcticon} from '@primer/components'
- import {Alert} from '@primer/octicons-react'
+ import {AlertIcon} from '@primer/styled-octicons'

function Example() {
    return (
-       <StyledOcticon icon={Alert} mr={2} />
+       <AlertIcon mr={2} />
    )
}
```
